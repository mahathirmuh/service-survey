üß™ Manual Testing Steps

  1. Initial Test:

    • Verify the "Remember me" checkbox is present and unchecked
    • Verify the checkbox is clickable and changes state
  2. Save Credentials Test:
    • Enter a valid username (e.g., andi.admin@example.com)
    • Enter the corresponding password (e.g., admin123)
    • Check the "Remember me" checkbox
    • Click "Sign In" to login successfully
  3. Persistence Test:
    • After successful login, logout or navigate away
    • Close the browser tab/window completely
    • Reopen the browser and navigate back to the login page
    • EXPECTED: Username should be pre-filled
    • EXPECTED: "Remember me" checkbox should be checked
  4. Clear Credentials Test:
    • Uncheck the "Remember me" checkbox
    • Login again with valid credentials
    • Close browser and reopen
    • EXPECTED: Username should be empty
    • EXPECTED: "Remember me" checkbox should be unchecked
